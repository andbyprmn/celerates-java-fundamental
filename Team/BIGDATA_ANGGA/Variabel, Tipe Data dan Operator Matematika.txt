#Variabel#
Variabel adalah kontainer untuk menyimpan nilai data.
Di dalam Java, terdapat beberapa perbedaan tipe variabel, contohnya:
-String --stores text, menggunakan double quote 
-Int --stores numbers tanpa decimal
-Float --stores number float, dengan decimal 
-Char --stores single karakter seperti 'a' atau 'B', menggunakan single quote
-Boolean --stores nilai dengan dua keadaan: benar atau salah
untuk mendeklarasikan variabel terdapat aturan yang harus diikuti, diantaranya:s
1. Nama variabel tidak boleh menggunakan kata kunci dari Java (reserved word) seperti if, for, switch, dll.
2. Nama variabel boleh menggunakan huruf, angka (0-9), garis bawah (underscore), dan symbol dollar ($), namun penggunaan garis bawah dan symbol lebih baik dihindari.
3. Nama variabel harus diawali dengan huruf kecil, karena Java menggunakan gaya CamelCase.
4. Apabila nama variabel lebih dari 1 suku kata, maka kata ke-2 dituliskan dengan diawali dengan huruf besar dan seterusnya, contoh namaVariabel.

#Tipe Data#
Tipe data dibagi menjadi dua grup:
-primitif tipe data = byte, int long, float, double, boolean dan char
-non primitif tipe data = string, array, class

#Operator Aritmatika#
Untuk melakukan operasi aritmatika pada Java terdapat beberapa jenis operator yang digunakan diantaranya :
+ untuk penambahan
- untuk pengurangan
* untuk perkalian
/ untuk pembagian
% untuk sisa hasil bagi
